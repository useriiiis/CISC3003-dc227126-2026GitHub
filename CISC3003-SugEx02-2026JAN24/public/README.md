## 如何查看网站

### 方式 1：在 Eclipse IDE 中
1. 打开项目
2. 右键点击 `public/cisc3003-sugEx02.html`
3. 选择 "Open With" → "Web Browser"
4. 选择内部浏览器或外部浏览器

### 方式 2：在外部浏览器中
1. 导航到 public 文件夹
2. 双击 `cisc3003-sugEx02.html`
3. 网站将在默认浏览器中打开

### 方式 3：使用文件资源管理器
1. 右键点击 `cisc3003-sugEx02.html`
2. 选择 "打开方式" → 选择浏览器（Chrome、Firefox、Edge）

## 使用的技术

- **HTML5**: 语义化标记和结构
- **CSS3**: 现代样式，使用 Flexbox 和 Grid
- **Google Fonts**: Raleway 字体家族
- **响应式设计**: 移动端友好布局
- **CSS 动画**: 平滑过渡和悬停效果

## 设计原则

1. **移动优先响应式设计**: 适配所有屏幕尺寸
2. **现代配色方案**: 专业的蓝色调
3. **排版层次**: 清晰的视觉层次和合适的字体大小
4. **留白空间**: 充足的间距以提高可读性
5. **用户体验**: 平滑滚动、悬停效果和直观导航
6. **可访问性**: 语义化 HTML 和适当的对比度
