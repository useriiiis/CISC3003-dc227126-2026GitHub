## Features

### 1. **Navigation Bar**
   - Sticky navigation that stays at the top while scrolling
   - Smooth scroll to different sections
   - Hover effects on menu items

### 2. **Hero Section**
   - Full-screen welcome banner with background image
   - Eye-catching call-to-action button
   - Smooth fade-in animation

### 3. **About Section**
   - Company story and values
   - Three feature cards highlighting key offerings
   - Professional image showcase

### 4. **Menu Section**
   - Grid layout displaying 6 coffee items
   - Individual cards with images, descriptions, and prices
   - Hover effects with smooth animations

### 5. **Contact Section**
   - Location, hours, and contact information
   - Interactive contact form
   - Clean, organized layout

### 6. **Footer**
   - Quick links navigation
   - Social media links
   - Student information (ID and Name)
   - Copyright notice

## How to View the Website

### Option 1: In Eclipse IDE
1. Open the project in Eclipse
2. Right-click on `cisc3003-sugEx02.html`
3. Select "Open With" → "Web Browser"
4. Choose Internal Web Browser or System External Browser

### Option 2: In External Browser
1. Navigate to the project folder
2. Double-click `cisc3003-sugEx02.html`
3. The website will open in your default browser

### Option 3: Using File Explorer
1. Right-click on `cisc3003-sugEx02.html`
2. Select "Open with" → Choose your preferred browser (Chrome, Firefox, Edge)
