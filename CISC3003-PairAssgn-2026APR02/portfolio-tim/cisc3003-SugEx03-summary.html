My Goals and Intended Learning Outcomes

Clear learning targets for this semester and beyond.

Goals

  • Build a complete multi-page personal website with clean UI.
  • Improve HTML structure and semantic tags (header, nav, section, footer).
  • Practice CSS layout (grid/flex), typography, spacing, and responsiveness.
  • Learn validation and debugging habits (validator + browser dev tools).
  • Develop a portfolio mindset: document learning, show progress.

Intended Learning Outcomes (ILO)

  • Design site navigation and page sections logically for stakeholders.
  • Write standards-compliant HTML and pass online validators.
  • Create external CSS stylesheets and maintain consistent theme.
  • Communicate my personal development plan and achievements clearly.
  • Publish work via GitHub and manage project files systematically.

My Reflection and Future Directions

What I learned and what I want to improve next.

What Went Well

I organized my project structure clearly and maintained a consistent visual theme across pages. The navigation and section layout help readers find information quickly.

Challenges

I needed time to refine spacing and make pages responsive. I learned to adjust CSS step-by-step and test in different browsers.

Next Steps

I want to add more interactive elements (basic JavaScript), improve accessibility (ARIA, contrast), and enrich the showcase content with real projects.
